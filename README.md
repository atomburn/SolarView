# EG4 Solar E-Ink Hub

This project contains two parts:

1. **The Dashboard (Web App):** A React application that generates deployment scripts and provides a visual dashboard.
   - Hosted at: [https://atomburn.github.io/SolarView/](https://atomburn.github.io/SolarView/)
   
2. **The Relay (Python Script):** A script generated by the Dashboard that runs via GitHub Actions to sync your inverter data to your E-Ink display.

## How to use

1. Go to the [Hosted Dashboard](https://atomburn.github.io/SolarView/).
2. Click **Hardware Setup**.
3. Select **Cloud Relay** -> **GitHub Actions**.
4. Follow the built-in Installation Manual to set up the data sync.
